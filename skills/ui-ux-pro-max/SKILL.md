# UI/UX Pro Max Skill

Design intelligence for professional UI/UX.

## Capabilities

### 1. Design Analysis
- Review existing designs
- Identify UX issues
- Suggest improvements
- Accessibility audit

### 2. Design Generation
- Create wireframes
- Generate mockups
- Design system components
- Responsive layouts

### 3. Platform Expertise
- **Web** - React, Vue, Angular, vanilla
- **Mobile** - iOS, Android, React Native
- **Desktop** - Electron, native apps
- **Design Tools** - Figma, Sketch integration

### 4. Best Practices
- Material Design
- Human Interface Guidelines
- WCAG accessibility
- Mobile-first responsive

## When to Use

✅ **USE when:**
- "Design a landing page"
- "Review this UI"
- "Make this more user-friendly"
- "Create a design system"
- "Fix these UX issues"
- "Is this accessible?"

## Design Principles

### Visual Hierarchy
- Size and scale
- Color and contrast
- Typography
- Spacing and layout

### User Flow
- Clear navigation
- Progressive disclosure
- Error prevention
- Feedback loops

### Accessibility
- Color contrast ratios
- Keyboard navigation
- Screen reader support
- Focus indicators

## Example Usage

```bash
# Analyze current design
uiux analyze --url http://localhost:3000

# Generate wireframe
uiux wireframe --type "dashboard" --platform web

# Create component
uiux component --name "Button" --style modern

# Check accessibility
uiux a11y --url http://example.com
```
