# MyVibe Skills

AI-powered web page creation and publishing.

## Capabilities

### 1. Static Site Generation
- Create HTML/CSS/JS sites
- Deploy to static hosting
- Support for templates

### 2. Framework Support
- **Vite** - Fast dev server and optimized builds
- **Next.js** - React framework with SSR/SSG
- **Static HTML** - Plain vanilla JS sites

### 3. Publishing
- One-click deploy
- Custom domains
- Version management

## When to Use

✅ **USE when:**
- "Create a website for X"
- "Build a landing page"
- "Deploy this project"
- "Make a portfolio site"
- "Create a blog"

## Workflow

1. **Design** - Describe what you want
2. **Generate** - AI creates the code
3. **Preview** - Local development server
4. **Publish** - Deploy to production

## Example

```bash
# Create a portfolio site
myvibe create portfolio --template modern

# Build with Vite
myvibe build --framework vite

# Deploy
myvibe publish --domain myportfolio.com
```

## Integration

Works with:
- **canvas** - Preview sites locally
- **browser** - Test interactions
- **github** - Version control
- **proactive-agent** - Auto-rebuild on changes
